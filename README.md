# Noticias Seguras (Flutter + NewsAPI)

App Flutter para consumir **NewsAPI** de forma **segura** con manejo de estados (cargando / error / vacío), **timeouts**, **retry exponencial**, **cache defensiva** y **sanitización** de entradas (OWASP).

## 1) Requisitos
- Flutter estable
- Android SDK/Emulador o dispositivo físico
- Una API key válida de https://newsapi.org

## 2) Instalación
```bash
flutter pub get
